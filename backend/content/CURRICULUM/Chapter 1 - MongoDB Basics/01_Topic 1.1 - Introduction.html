📗 Topic 1.1: Introduction & Key Concepts

📌 What is MongoDB?

MongoDB is a NoSQL database used to store data in JSON-like documents. Unlike traditional SQL databases, MongoDB offers:

  • No tables (uses collections)
  • No fixed schema
  • Flexible structure

📌 Key Concepts

Concept Description
Database Collection of data
Collection Group of documents (similar to a table)
Document JSON-like record
Field Key-value pair

📌 Understanding Structure

Level → Example

Database → schoolDB

Collection → students

Document → One student record

📌 Example Document

{
  id: 44,
  name: "Diana Prince",
  age: 23,
  grade: "C",
  marks: 65,
  department: "History"
}

📌 How It Looks in MongoDB

schoolDB
   └── students
