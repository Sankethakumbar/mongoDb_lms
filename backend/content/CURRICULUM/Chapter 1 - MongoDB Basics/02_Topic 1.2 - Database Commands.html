📗 Topic 1.2: Database Commands

📌 Common MongoDB Commands

Command Description
show dbs Show all databases
use mydb Select or create database
db Show current database
show collections Show collections in current DB
db.help() Show available commands
db.dropDatabase() Delete current database

📌 Examples

🔹 Show All Databases

show dbs

🔹 Select / Create Database

use schoolDB

🔹 Check Current Database

db

🔹 Show Collections

show collections

🔹 Drop Database

db.dropDatabase()

📌 Step-by-Step Example

show dbs
use schoolDB
db
show collections

👉 This will:

  1. Show all databases
  2. Create/select schoolDB
  3. Confirm current database
  4. Show collections
