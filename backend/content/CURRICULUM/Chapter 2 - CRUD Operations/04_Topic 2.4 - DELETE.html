📗 Topic 2.4: DELETE (Remove Data)

📌 MongoDB DELETE Statement

The delete operation is used to remove documents from a collection permanently.

📌 Delete Commands

CommandDescription
deleteOne()Delete one document
deleteMany()Delete multiple documents

📌 Syntax

db.collection.deleteOne(filter)
db.collection.deleteMany(filter)

📌 Delete One Example

db.students.deleteOne({ name: "Diana Prince" })

📌 Delete Many Example

db.students.deleteMany({ marks: { $lt: 60 } })

📌 SQL vs MongoDB

SQLMongoDB
DELETE FROM students WHERE name='Diana'deleteOne()
DELETE multiple rowsdeleteMany()

🎯 Topic 2.4 Summary

  • DELETE = Remove data
  • Use deleteOne() / deleteMany()
  • Be careful — deletion is permanent ⚠️
