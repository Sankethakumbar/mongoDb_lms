📗 Topic 2.3: UPDATE (Modify Data)

📌 MongoDB UPDATE Statement

The update operation is used to modify existing documents in a collection.

📌 Update Commands

CommandDescription
updateOne()Update one document
updateMany()Update multiple documents
$setUpdate specific fields safely

📌 Syntax

db.collection.updateOne(filter, update)
db.collection.updateMany(filter, update)

📌 Update One Example

db.students.updateOne(
  { name: "Diana Prince" },
  { $set: { marks: 70 } }
)

📌 Update Many Example

db.students.updateMany(
  { department: "Computer Science" },
  { $set: { status: "active" } }
)

📌 Important Note ⚠️

Without $set, the whole document gets replaced ❗

Always use $set to safely update specific fields.

📌 SQL vs MongoDB

SQLMongoDB
UPDATE students SET marks=70updateOne()
UPDATE multiple rowsupdateMany()
