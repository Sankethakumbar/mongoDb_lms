📗 Topic 2.1: CREATE (Insert Data)

📌 MongoDB INSERT Statement

The insert operation is used to add data to a collection. The data is stored as documents.

📌 Insert Commands

CommandDescription
insertOne()Insert one document
insertMany()Insert multiple documents
save()Insert or update document
insert()Deprecated ❌

📌 Syntax

db.collection.insertOne(document)
db.collection.insertMany([doc1, doc2])
db.collection.save(document)

📌 Insert One Example

db.students.insertOne({
  id: 44,
  name: "Diana Prince",
  age: 23,
  grade: "C",
  marks: 65,
  department: "History"
})

📌 Insert Many Example

db.students.insertMany([
  {
    id: 45,
    name: "Ethan Hunt",
    age: 20,
    grade: "B",
    marks: 82,
    department: "Computer Science"
  },
  {
    id: 46,
    name: "Bruce Wayne",
    age: 24,
    grade: "A",
    marks: 90,
    department: "Business"
  }
])

📌 Save Example

db.students.save({
  _id: 1,
  name: "Tom",
  age: 28
})

📌 How save() Works

ConditionResult
_id existsUpdates document
_id not existsInserts document

📌 SQL vs MongoDB

SQLMongoDB
INSERT INTO studentsinsertOne()
INSERT multiple rowsinsertMany()
UPDATE using IDsave()

🎯 Topic 2.1 Summary

  • CREATE = Insert data
  • Use insertOne() / insertMany()
  • save() can insert or update
  • _id is unique
