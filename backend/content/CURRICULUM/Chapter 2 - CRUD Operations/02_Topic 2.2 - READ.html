📗 Topic 2.2: READ (Find Data)

📌 MongoDB FIND Statement

The find() method is used to retrieve data from a collection. The result is returned as a cursor (result set).

📌 Find Commands

CommandDescription
find()Get all documents
find(query)Get filtered data
find({}, projection)Select specific fields

📌 Syntax

db.collection.find(query, projection)

📌 Select All Example

db.students.find()

📌 Select Specific Fields Example

db.students.find({}, {name: 1, marks: 1})

📌 Condition Example (WHERE)

db.students.find({marks: {$gt: 70}})

📌 SQL vs MongoDB

SQLMongoDB
SELECT * FROM studentsdb.students.find()
SELECT name FROM studentsdb.students.find({}, {name:1})
SELECT * WHERE marks > 70db.students.find({marks: {$gt:70}})
