📌 Stages Table

StageMeaningSQL Equivalent
$matchFilter dataWHERE

$projectSelect fieldsSELECT
$sortSort dataORDER BY
$limitLimit resultsLIMIT

📌 Syntax for Each Stage

🔹 $match — Filter

{ $match: { condition } }

🔹 $group — Group

{
  $group: {
    _id: "$field",
    result: { operator }
  }
}

🔹 $project — Select Fields

{ $project: { field1: 1, field2: 1 } }

🔹 $sort — Sort Results

{ $sort: { field: 1 } }   // 1 = Ascending, -1 = Descending

🔹 $limit — Limit Count

{ $limit: 5 }

🔹 $skip — Skip Records

{ $skip: 10 }

🎯 Topic 4.2 Summary

  • Each stage transforms the data
  • Stages run in order through the pipeline
  • You can combine as many stages as needed
