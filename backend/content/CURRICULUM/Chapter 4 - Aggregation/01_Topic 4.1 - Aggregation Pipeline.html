📗 Topic 4.1: Aggregation Pipeline

📌 What is Aggregation?

Aggregation is used to process and analyze data in MongoDB. It is similar to GROUP BY in SQL.

📌 Why use Aggregation?

  • Calculate totals
  • Find averages
  • Group data
  • Analyze large datasets

📌 Definition

Aggregation works using a pipeline of stages. Data flows step-by-step and gets transformed at each stage.

📌 Syntax

db.collection.aggregate([
  { $stage1: { ... } },
  { $stage2: { ... } },
  ...
])

📌 Basic Visualization

[ Data ] → $match → $group → $sort → $limit → [ Result ]

🎯 Topic 4.1 Summary

  • Aggregation = Analyze + Process Data
  • Uses a pipeline of stages
  • Similar to GROUP BY in SQL
