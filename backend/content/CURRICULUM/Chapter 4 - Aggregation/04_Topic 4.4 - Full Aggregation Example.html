📗 Topic 4.4: Full Aggregation Example

📌 Sample Data (students collection)

[
  { name: "A", department: "CS", marks: 80 },
  { name: "B", department: "CS", marks: 70 },
  { name: "C", department: "History", marks: 50 },
  { name: "D", department: "History", marks: 90 }
]

📌 Full Query

db.students.aggregate([
  { $match: { marks: { $gt: 60 } } },
  {
    $group: {
      _id: "$department",
      avgMarks: { $avg: "$marks" }
    }
  },
  { $sort: { avgMarks: -1 } },
  { $limit: 2 }
])

📊 Step-by-Step Execution

🔹 Step 1: $match (Filter)

Condition: marks > 60

Result:

A (80), B (70), D (90)

🔹 Step 2: $group (Group by department)

CS → (80 + 70) / 2 = 75

History → 90

🔹 Step 3: $sort (Descending)

History → 90

CS → 75

🔹 Step 4: $limit

Top 2 results

📌 Final Output

[
  { "_id": "History", "avgMarks": 90 },
  { "_id": "CS", "avgMarks": 75 }
]
