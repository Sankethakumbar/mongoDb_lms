📗 Topic 4.3: Aggregation Operators

📌 Definition

Aggregation operators are used inside $group to calculate values across grouped documents.

📌 Operators Table

OperatorMeaningSQL Equivalent
$sumTotal / CountSUM()
$avgAverageAVG()
$maxMaximum valueMAX()
$minMinimum valueMIN()
$countCount documentsCOUNT()
$pushCollect into array—

📌 Usage Examples

🔹 Sum of Marks

db.students.aggregate([
  {
    $group: {
      _id: "$department",
      totalMarks: { $sum: "$marks" }
    }
  }
])

🔹 Average Marks

db.students.aggregate([
  {
    $group: {
      _id: "$department",
      avgMarks: { $avg: "$marks" }
    }
  }
])

🔹 Maximum Marks

db.students.aggregate([
  {
    $group: {
      _id: "$department",
      topScore: { $max: "$marks" }
    }
  }
])

🔹 Count Students Per Department

db.students.aggregate([
  {
    $group: {
      _id: "$department",
      studentCount: { $sum: 1 }
    }
  }
])

🎯 Topic 4.3 Summary

  • Operators go inside $group
  • $sum: 1 is used to count documents
  • Works like SQL aggregate functions
