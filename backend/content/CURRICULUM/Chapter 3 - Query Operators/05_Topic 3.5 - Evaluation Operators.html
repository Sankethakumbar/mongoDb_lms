📗 Topic 3.5: Evaluation Operators

📌 Definition

Evaluation operators are used to search and match text patterns inside field values.

📌 Operators Table

OperatorMeaning
$regexPattern matching using regular expressions
$textFull text search (requires text index)
$whereMatch using JavaScript expression

📌 Examples

🔹 Names Starting With "D"

db.students.find({
  name: { $regex: "^D" }
})

👉 Returns Diana Prince (starts with D)

🔹 Names Containing "Hunt"

db.students.find({
  name: { $regex: "Hunt" }
})

🔹 Case-Insensitive Search

db.students.find({
  name: { $regex: "diana", $options: "i" }
})

👉 $options: "i" makes it case-insensitive

📌 Common Regex Patterns

PatternMeaning
^DStarts with D
e$Ends with e
HuntContains "Hunt"
^[A-Z]Starts with any capital letter
