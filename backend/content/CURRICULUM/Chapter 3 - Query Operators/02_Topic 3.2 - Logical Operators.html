📗 Topic 3.2: Logical Operators

📌 Definition

Logical operators combine multiple conditions, just like AND / OR in SQL.

📌 Operators Table

OperatorMeaning
$andAll conditions must be true
$orAny condition can be true
$notNegates condition
$norNone of the conditions should be true

📌 Examples

🔹 AND

db.students.find({
  $and: [
    { marks: { $gt: 70 } },
    { grade: "A" }
  ]
})

🔹 OR

db.students.find({
  $or: [
    { marks: { $lt: 60 } },
    { grade: "C" }
  ]
})

🔹 NOT

db.students.find({
  marks: { $not: { $gt: 80 } }
})

📌 SQL vs MongoDB

SQLMongoDB
WHERE A AND B$and
WHERE A OR B$or
WHERE NOT A$not

🎯 Topic 3.2 Summary

  • Combine multiple conditions
  • Use $and, $or, $not
