📗 Topic 3.4: Element Operators

📌 Definition

Element operators are used to check field existence or data type in a document.

📌 Operators Table

OperatorMeaning
$existsCheck if field exists or not
$typeCheck the data type of a field

📌 Examples

🔹 Check Field Exists

db.students.find({
  marks: { $exists: true }
})

👉 Returns only documents that have the marks field.

🔹 Check Field Does NOT Exist

db.students.find({
  marks: { $exists: false }
})

🔹 Check Data Type

db.students.find({
  age: { $type: "int" }
})

👉 Returns documents where age is stored as an integer.

📌 Common Type Values

Type StringMeaning
"string"Text value
"int"Integer number
"double"Decimal number
"bool"True / False
"array"Array value

🎯 Topic 3.4 Summary

  • Used to check structure of data
  • $exists — check if field is present
  • $type — check the field's data type
