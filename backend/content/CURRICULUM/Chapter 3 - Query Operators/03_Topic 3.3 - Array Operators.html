📗 Topic 3.3: Array Operators

📌 Definition

Array operators are used when fields contain arrays or when matching against a list of values.

📌 Operators Table

OperatorMeaning
$inMatches any value in array
$ninNot in array
$allMatches all values in array

📌 Examples

🔹 IN Operator

db.students.find({
  department: { $in: ["CS", "History"] }
})

🔹 NOT IN

db.students.find({
  department: { $nin: ["Math"] }
})

📌 SQL vs MongoDB

SQLMongoDB
IN ('CS', 'History')$in
NOT IN ('Math')$nin

🎯 Topic 3.3 Summary

  • Used for matching multiple values
  • Works like SQL IN / NOT IN
