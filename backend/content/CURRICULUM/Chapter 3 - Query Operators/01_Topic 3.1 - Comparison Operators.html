📗 Topic 3.1: Comparison Operators

👉 Query operators are used to filter data (like WHERE in SQL)

📌 Definition

Comparison operators are used to compare values in fields.

📌 Operators Table

OperatorMeaning
$gtGreater than
$ltLess than
$gteGreater than or equal
$lteLess than or equal
$eqEqual
$neNot equal

📌 Examples

🔹 Greater Than

db.students.find({ marks: { $gt: 70 } })

🔹 Less Than

db.students.find({ marks: { $lt: 60 } })

🔹 Greater Than or Equal

db.students.find({ marks: { $gte: 80 } })

🔹 Not Equal

db.students.find({ grade: { $ne: "A" } })

📌 SQL vs MongoDB

SQLMongoDB
marks > 70{marks: {$gt: 70}}
marks < 60{marks: {$lt: 60}}
marks >= 80{marks: {$gte: 80}}

🎯 Topic 3.1 Summary
